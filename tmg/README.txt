🚀 Como rodar o EsperançaBot + Painel Admin

1️⃣ Instale Python 3.10 ou superior

2️⃣ Instale as dependências:
pip install -r requirements.txt

3️⃣ Coloque o arquivo firebase_config.json na mesma pasta dos arquivos

4️⃣ Execute o bot Telegram:
python main.py

5️⃣ Execute o painel admin:
python panel.py
Acesse no navegador: http://127.0.0.1:5000
Senha padrão do painel: senhaadmin (edite dentro do arquivo panel.py)

6️⃣ Para rodar na nuvem:
- Use Railway.app, Render.com ou VPS
- Funciona perfeitamente

7️⃣ Personalização:
- Edite a chave PIX no main.py
- Edite a senha do painel no panel.py
- Edite o token do bot no main.py

8️⃣ 🚀 Pronto para ganhar dinheiro no Telegram!
